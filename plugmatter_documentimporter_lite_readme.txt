== Description ==

<h3>What is Document Importer Plugin?</h3>

[youtube https://www.youtube.com/watch?v=DMhBfXIRrlg]

Plugmatter Document Importer is a WordPress plugin that lets you effortlessly import your document (.DOCX) directly into WordPress Editor. 

Document Importer is a WordPress plugin that allows you to import .docx documents from Microsoft Word, Google Docs or your computer into your WordPress editor — exactly as they are — with just one click.

It retains everything from images to bullet points, making your publishing experience a breeze. How? Magic.

_**Note:**</strong> The plugin only supports .DOCX file format and not .DOC file format._

<h3>Who is It For?</h3>

The Document Importer plugin is for WordPress users who love to write their blog posts in Microsoft Word or Google Docs rather than the WordPress editor. 

It is also immensely useful for:

* Professional blogger that write long form posts
* Photographer that uploads multiple images in each post

* Receive and publish articles from freelancers sent to you in the .docx format

Simply because it allows you to:

* Write more and grow your business instead of spending time on content formatting.

* Enhance the overall site appearance by posting amazingly well-formatted articles.


<h3>What It Does?</h3>
The plugin ensures that all the content formatting in the document remains in tact when it gets imported. In short, it:

**Retains Links to the Content:** With Document Importer plugin, you longer to have re-do all the hyperlink addition in WordPress Editor.

**Uploads and Resizes Images:** The plugin also automatically uploads and resizes images in WordPress exactly how they are placed in the document. It also automatically saves the images in Media Library for future use.

**Retains Format Content:** It retains the entire content formatting including its subheadings, quotes, font color, so you don’t have to format it again.

**Retains Bullet Points:** The plugin allows you to keep away from dealing with frustrating bullet points in WordPress - once and for all.

== Installation ==

You can install Document Importer by Plugmatter in different / traditional ways:

WordPress Search:
----

1. Use WordPress Plugin Search in your WordPress
2. Look for "Document Importer by Plugmatter"
3. Click to Install and Activate
4. A new Document Importer by Plugmatter will appear in the "Settings" in the left menu

Download & Install:
----

1. You can download the ".zip" file from the top right of this page on to computer
2. Go to your Plugins section and click "add new"
3. Select the tab "Add"
4. Click "Upload" to upload and install the plugin from your computer / desktop
5. Once installed, activate the plugin
6. A new Document Importer by Plugmatter will appear on your Admin column

FTP:
----

1. You can also install the plugin using the FTP
2. Simply Download the ".zip" file of Plugin from top right of this page
3. Login to your WordPress FTP
4. Upload the ".zip" file in your wp-content > Plugins
5. Login to your WordPress and Activate the Plugin
6. A new Document Importer by Plugmatter will appear on your Admin column


== Frequently Asked Questions ==
<h3>What file format does the plugin import?</h3>
Document Importer by Plugmatter only imports .docx file format as of now. So before importing the document, make sure that it is in .docx format.

<h3>How is this different from "Paste from Word"?</h3>
Document Importer is different from Paste from Word in various ways. Our plugin uses advanced XSLT parser to convert a word file into a HTML file, which retain all the formatting when imported. 

Our plugin not only imports images with the content but also adds them to media gallery, automatically. This allows you to use the images for future purposes.

<h3>Does it also import header and footer from the document?</h3>
No, the Document Importer plugin does not import header and footer from the document.

<h3>What else can the plugin not import?</h3>
The plugin cannot import graphics generated with Microsoft Word or Google Docs. For any such graphic or drawing, it is advised to convert it in an image and then import it.

<h3>Where do I go for technical support?</h3>
You can send us an email at support@plugmatter.com with your query and our dedicated support team will get back to you as soon as possible.

<h3>Can you help me with API Setup of Google Drive and Dropbox?</h3>
Of course, we can help with your API Setup. You can follow our Document Importer by Plugmatter user guide to learn how to do it. However, if you have any concerns concerns, you can always email us at support@plugmatter.com

<h3>Will Document Importer by Plugmatter work on WordPress.com?</h3>
No. Document Importer by Plugmatter is not compatible with WordPress.com (the hosted version of WordPress). This is because WordPress.com doesn't allow its users to install any third-party plugins.
